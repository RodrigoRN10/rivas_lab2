Semana 04

Temas: CSS - Hosting

Abril 11-12, 2023 Jane Doe

Séptima sesión: para esta clase se puso en práctica nuestras habilidades para recordar lo aprendido durante las primeras semanas, como ejercicio se dio crear una web con códigos básicos como las de head, bodyl, ul, li, div, p y acompañados de algunos atributos con href, class y id, pero para que esta página tenga algo de estética se utilizó CSS para darle pequeñas modificaciones a la página con width, margin, overflow, float, padding, background-color y height.

Octava sesión: Durante esta clase, se nos iba explicando cómo nuestra página web podría ser encontrada. Primero tendríamos que encontrar una página dedica para el hosting como es WIX, cuya función es alojar y almacenar los datos e información de nuestra página. Luego, darle un nombre para ese caso estaría lo que es dominio, que es con lo que las personas podrán buscarte por internet. Por último, estaría subir tus archivos al hosting y publicarlos con el domino de tu elección.
